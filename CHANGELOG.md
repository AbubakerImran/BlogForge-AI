## [0.1.0] - 2024

### Added
- Initial release of BlogForge AI
- Next.js 14 with App Router and TypeScript
- Rich text editor powered by Tiptap
- AI-powered post summaries using Groq API (LLaMA 3.3 70B)
- Full authentication system with NextAuth.js (Google OAuth + credentials)
- PostgreSQL database with Prisma ORM
- Admin dashboard with analytics
- Blog post management (create, edit, delete, publish)
- Category and tag system
- Newsletter subscription system with Resend integration
- Full-text search functionality
- Analytics dashboard with charts
- Page view tracking
- SEO optimization (meta tags, Open Graph, sitemap, robots.txt)
- Dark mode support with next-themes
- Responsive design with Tailwind CSS
- shadcn/ui component library
- Database seeding with sample data
- Email/password authentication with bcrypt hashing
- Role-based access control (USER, ADMIN)
- Magazine-style homepage layout
- Individual blog post pages with table of contents
- Social sharing buttons
- KPI cards (posts, views, subscribers, revenue)
- Newsletter subscriber management with CSV export
- Category and tag filtered views
- Author profiles
- Featured posts
- Reading time calculation
- Ad placeholder components
- Contact page
- About page
- Privacy policy page
- Terms of service page

### Tech Stack
- **Framework:** Next.js 14
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS 3
- **Database:** PostgreSQL with Prisma ORM
- **Authentication:** NextAuth.js v4
- **AI:** Groq API (LLaMA 3.3 70B)
- **Editor:** Tiptap
- **Charts:** Recharts
- **Email:** Resend
- **UI Components:** shadcn/ui
- **Icons:** Lucide React
